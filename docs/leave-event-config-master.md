## 1) Earned Leave (EL) - Event Category: Leave

- Authorized: YES
- Consider as work hours: YES
- Has balance: YES
- Credit from Over Time: NO
- Allow Balance Entry: YES
- Allow Event Opening Rule: YES
- Allow Auto Credit Rule: YES
- Allow Hourly: NO (half-day EL allowனா YES)
- Allow Datewise: YES
- Allow WeekOff Selection: NO
- Allow Holiday Selection: NO
- Applicable for Regularization: NO
- Allow Different Leave Period: YES
- Allow Event Change: NO
- Validation Remarks Mandatory: NO (policy-based)
- Event Entry Form: Default
- Auto Credit Engine: Default
- Encashment: Default
- Leave Deduction while in F&F: NO

- Save பண்ணும் முன் overlap rules and approval workflow check செய்யவும்.
